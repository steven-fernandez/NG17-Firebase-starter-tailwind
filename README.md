# Angular 17 / Tailwind / Firebase Starter 

# Instructions

1. Clone, cd into the project and npm install
2. Open your project using your editor.
3. Rename example.environment.ts to environment.ts and add your firstbase details
4. ng serve -o

Project has firebase AUTH
Sign-in, Sign-up, Forgotten password, Logout, verify email, guards and dashboard, tailwind and daisy UI.


This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.1.0.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
